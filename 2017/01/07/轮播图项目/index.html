轮播图项目总结

2017/01/07

本文于436天之前发表，文中内容可能已经过时。

轮播图项目总结：

切换方式: 上下滑动 左右滑动 直接显示/隐藏

直接显示/隐藏样式:

wrapper:

  • position: absolute;
  • display: none;
  • 配置 slider-active

盒子

按钮(上一张/下一张)

  • 这里有个坑:
    左右两个按钮除了 left 样式不同之外,其他都相同,这样我们可以先给所有的 btn 都设置一样的定位和样式,在通过独立修改右面 btn 的right=0就可以了,但是事实上并不可以,应为如果position=absolute;left 和 right 都有的话,浏览器会忽略 right,只用 left 作为定位,所以我们应该给右面的 btn 先设置left=auto,再right=0,可以解决此问题

js 部分

  1. var timer = null;
  2. function autoMove
  3. 获取 ul, moveWidth, btn, slider
  4. autoMove 先实现自动轮播功能
  5. 用 index 来获取当前轮播的第几个图片
  6. function changeIndex 通过 index 来讲 span 实现同步轮播,先把所有的 span 的 class 全部为空,再把 span[index]是 active

bug

边界时由于 index 一直自加,span 的 className 赋值就会出错,找不到对应的 span

刚开始点击 leftbtn, 第一张图片的上一张为空解决办法:if(你得当前位置 left 是 0)时 点击上一页按钮,直接把 left 变成最后一张的位置,记得加”px”,
还有把 index 设置成最后一张的值

leftbtn 点击会与 autoMove 函数争抢解决办法:每次都在最上方 clear 一下, 但是定时器就得用 setTimeout()
在最后在开一个定时器

疯狂点击,轮播图崩溃了解决办法 上个锁

给所有 span 附上点击事件,会有闭包,用立即执行函数,参数写 myindex,防止和全局 index 污染
