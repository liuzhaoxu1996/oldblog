获取窗口属性，获取 dom 尺寸

2017/06/01

获取窗口属性，获取 dom 尺寸

  • 正常浏览器 ==ie9 以下不兼容==
    • pageXOffset() pageYOffset() 横向滚动条/竖向滚动条 滚动距离
  • ie8 和 ie8 以下的浏览器使用的方法
    • document.body.scrollTop/Left ==ie8 ie5 ie4==
    • document.documentElement.scrollLeft/Top ==ie7 ie6==
  • 封装一个 getScrollOffsets 都兼容的方法

可视区窗口

正常浏览器方法

  • window.innerWidth 可视区窗口的宽
  • window.innerHeight 可视区窗口的高

ie8 及 ie8 以下(标准模式：该版本正常的渲染模式)

  • document.documentElement.clientWidth/clientHeight

ie8 及 ie8 以下(怪异模式/混杂模式：放弃新版本浏览器的渲染模式，使用以前浏览器的渲染模式),把<!DOCTYPE html> 去掉就是怪异模式

  • document.body.clientWidth/clientHeight

  • compat 是兼容的意思，如果是标准模式，compatMode 是”css1Compat”,如果是怪异模式，则是”BackCompat”

封装兼容性方法：返回浏览器的

查看元素的几何尺寸

  • domEle.getBoundingClientRect() 返回一个对象，包含 top right bottom left width height 六个属性，兼容性很好，返回的结果不是实时的,是一个静态写照。（不常用），left 和 top 代表该元素左上角的 X/Y 坐标，right 和 bottom 代表元素右下角的 X 和 Y 坐标
  • dom.offsetWidth() 视觉的宽度(比如一个 div 的宽为 100px，padding 为 100px，则返回 300px)
  • dom.offsetHeight() 视觉的高度
  • dom.offsetLeft() 相对于最近有定位父级的 left 值，如果父级没有定位，返回相对文档的坐标
  • dom.offsetParent() 返回最近有定位的父级节点。
  • 练习：封装求元素相对于文档的坐标 getElementPosition()方法
  • 让滚动条滚动
    • window 上有三个方法：
      • window.scroll(x, y) 让滚动条滚动到(x,y)点
      • window.scrollTo(x, y) 同上 一样
      • window.scrollBy(x, y) 滚动(x,y)距离，具有累加性

demo：阅读器

脚本化 CSS(重点)

  • div.style.width/[“width”] 就这个可以写入属性
    • 只能获取 div 的行间 style 样式的 width 属性，
    • 可读可写(内联属性值不能读写)
    • 获取不了内联样式里的属性
    • 建议：复合属性最好拆解，组合属性用小驼峰式写法
    • 建议：写入的值必须是字符串格式**
    • 建议：注意保留字：float -> cssFloat
  • window.getComputedStyle(div, null) 获取一个样式表；与 style 样式表区别，它有默认值。
    • 返回的都是绝对值，没有相对值
    • ie8 及 ie8 以下不兼容
  • ele.currentStyle ie8 及 ie8 以下 查询样式
